# 🧪 مملكة العلم - Kingdom of Science

## WhatsApp RPG Bot

بوت واتساب RPG مستوحى من شخصية سينكو إيشيغامي من أنمي Dr. Stone

### 🚀 المميزات

- **16 لعبة متنوعة** - ألعاب تعليمية وترفيهية
- **نظام أدوار** - عالم 🧪، محارب ⚔️، جامع 🌿، قائد 👑
- **نظام التطور** - 5 مستويات لكل دور
- **نظام الموارد** - حجر، حديد، حمض نيتريك، زجاج
- **نظام العشائر** - إنشاء وإدارة عشائر جماعية
- **تحميل الوسائط** - يوتيوب، تيك توك، انستغرام، سبوتيفاي وأكثر
- **أدوات متقدمة** - ذكاء اصطناعي، ترجمة، طقس، صور

### 📋 الأوامر الأساسية

#### 🎮 الألعاب
```
.انضم              - الانضمام للعبة
.حالتي              - عرض حالتك الشخصية
.لوحة الشرف        - أفضل 10 لاعبين

.تحدي علمي         - سؤال علمي متعدد الخيارات
.لغز               - لغز عربي
.كلمة مشفرة        - كلمة مقلوبة الحروف
.صح أم خطأ         - سؤال صح/خطأ
.سؤال سريع         - سؤال حظ
.عجلة الحظ         - عجلة الحظ اليومية
.قصة               - مهمة قصصية
.يومي              - مهمة يومية
.رياضيات [مستوى]   - أسئلة رياضية
.رهان [رقم]        - رهان النرد
.حدث غامض          - حدث عشوائي
.استكشاف [منطقة]   - استكشاف الخريطة
.معركة [رقم]       - معركة PvP
.بطولة             - بطولة جماعية
.عشيرتي            - إدارة العشيرة
```

#### 📥 التحميل
```
.يوتيوب [رابط]     - تحميل فيديو يوتيوب
.صوت يوتيوب [رابط] - تحميل MP3 من يوتيوب
.بحث يوتيوب [اسم]  - بحث في يوتيوب
.تيك توك [رابط]    - تيك توك بدون علامة
.انستغرام [رابط]   - ريلز وصور انستغرام
.فيسبوك [رابط]     - فيديو فيسبوك
.تويتر [رابط]      - فيديو تويتر/X
.سبوتيفاي [رابط]   - تحميل من سبوتيفاي
.ساوندكلاود [رابط] - تحميل من ساوندكلاود
.كلمات [اسم]       - كلمات أغنية
```

#### 🛠️ الأدوات
```
.ذكاء [سؤال]       - GPT عربي
.ترجم [نص]         - ترجمة عربي
.ترجم en [نص]      - ترجمة لإنجليزي
.نطق [نص]          - نص إلى صوت
.طقس [مدينة]       - حالة الطقس
.صورة [بحث]        - بحث صورة
.اخبار [موضوع]     - آخر الأخبار
```

.عداد [ثواني]       - عداد تنازلي
.عكس [نص]           - عكس النص
.لغة سرية [نص]     - تشفير بالإيموجي
